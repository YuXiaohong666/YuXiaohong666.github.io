建立JDK的环境

下载与安装JDK

配置JDK环境变量

安装好JDK之后，需进行环境变量的配置。

  1. 右键“此电脑”，选择“属性”，点击“高级系统设置”，在弹出的面板点击“环境变量按钮”。
  2. 在“系统变量”选项组中，单击新建按钮，输入变量名JAVA_HOME，变量值为JDK的安装路径。
  3. 再次新建系统变量，变量名为CLASSPATH，变量值为**.;%JAVA_HOME%\lib;**
  4. 在环境变量对话框中，选择系统变量中的Path，新增变量值如下图

验证JDK是否配置

打开CMD，输入java –version，屏幕上会显示JDK的版本信息；再次使用javac命令，出现提示信息，表示JDK配置成功。

下载与安装Tomcat

从Apache官网可以下载相应版本的文件。

  1. 在环境变量对话框的系统变量中，新建系统变量CATALINA_HOME，并将值设置为Tomcat的安装路径。
  2. 修改系统变量CLASSPATH，新建值**%CATALINA_HOME%\lib;**。

搭建JavaWeb开发环境

下载与安装Eclipse

在Eclipse中配置JDK

  1. 从菜单栏选择Window->Preferences（首选项）命令，在弹出的Preferences对话框的左侧选择Java->Installed JREs（已安装的JRE），在右侧单击Add（添加）按钮，在弹出的Add JRE对话框中选择Standard VM选项。
  2. 单击Next选项，进入JRE Definition设置界面，单击Directory按钮，在弹出的界面中指定JRE的安装路径，也可在JRE home文本框中输入JRE安装路径。此处，通过Directory按钮找到安装的JDK版本的JRE安装路径。确定后，JRE home、JRE name和JRE system libraries会自动添加进来，单击Finish按钮完成添加。

在Eclipse中配置Tomcat

  1. 在Eclipse的菜单栏中选择Window->Preferences命令，弹出Preferences对话框，在左侧选择Server->Runtime Environment，单击右侧的Add（添加）按钮，选择Apache->Apache Tomcat v9.0，并选中Create a new local server复选框。
  2. 单击Next按钮，在Tomcat Server界面中，选择Tomcat的安装路径，单击Browse按钮选择Tomcat的安装路径，在JRE下拉列表中可用默认的Workbench default JRE，也可选择之前添加的JRE，单击Finish按钮完成。
  3. 配置完成后，在Eclipse主界面下方的Servers选项卡中，就可以看见添加的Tomcat v9.0 Server at localhost服务器了。
  4. 双击配置好的Tomcat v9.0 Server at localhost 服务器，在Overview下的Server Locations中选中Use Tomcat installation(takes control of Tomcat installation)单选按钮，并将Deploy path文本框中的内容修改为webapps，修改相关内容后保存设置。
